Meta tags

Aula 2 do curso programação web explicando as <meta tags>

<meta charset="UTF-8">

Meta tag compativel com acentuações.

<meta http-equiv="X-UA-Compatible" content="IE=edge">

Meta tag compativel com navegador window explore.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Meta tag compativel com o dispositivo do usuário (notbook, celular...)

<meta name="description" content="Esse é o meu primeiro site.">

Meta tag para uma pequena descrição do site.

<meta name="keywords" content="Palavras-chave.">

Meta tag para usar palavras-chave separadas por virgulas.

<meta name="author" content="Igor Gazineo">

Meta tag para identificar autor do site.
